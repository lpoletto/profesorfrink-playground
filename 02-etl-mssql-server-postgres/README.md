Crear y acceder a la carpeta

```sh
mkdir spark_jars
cd spark_jars
```

Descargar los JARs necesarios para conectarse a las DB.

MSSQL Server
```sh
wget https://repo1.maven.org/maven2/com/microsoft/sqlserver/mssql-jdbc/12.6.1.jre11/mssql-jdbc-12.6.1.jre11.jar 
``` 

PostgreSQL

```sh
wget https://repo1.maven.org/maven2/org/postgresql/postgresql/42.5.2/postgresql-42.5.2.jar
```
